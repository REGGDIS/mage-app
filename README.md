# MageApp (Gestión de Riesgos)

Proyecto con arquitectura de **tres capas**:

- **Backend**: `/mageapp-backend`  
  Node.js (Express), MySQL, JWT.
- **Frontend**: `/mageapp-frontend`  
  React + Vite + Bootstrap.
- **Base de datos**: MySQL con vistas `modelodevalor`, `matrizderiesgo`, `mapaderiesgos`.
